Código basado en el Tocani_SweepData.

Este código esta diseñado para ser usado en un Arduino Mega 2560.

Se controlan 4 robots para la toma de datos. La toma de datos se hará a través
de una Raspberry Pi Zero W, por medio de comunicación MQTT. Sin embargo, el movimiento
del robot de 2 ejes de cada piso se maneja de forma externa.
